# Project Title

Projects of PostgreSQL

## Description

SQL lessons projects in dvdrental database at Patika Academy

### Project 1
```
Perform the following query scenarios using the dvdrental sample database.

  - Sort the data in the title and description columns of the film table.
  - Sort the data in all columns in the film table, with the film length being greater than 60 AND less than 75.
  - Sort the data in all columns in the film table as rental_rate 0.99 AND replacement_cost 12.99 OR 28.99.
  - What is the value in the last_name column of the customer whose value is 'Mary' in the first_name column in the customer table?
  - Sort the data in the film table whose length is NOT greater than 50 and whose rental_rate value is NOT 2.99 or 4.99.
```

### Project 2
```
Perform the following query scenarios using the dvdrental sample database.

  - Sort the data in all columns in the film table, provided that the replacement cost value is greater than or equal to 12.99 and less than 16.99 (use the BETWEEN - AND structure).
  - Sort the data in the first_name and last_name columns in the actor table, provided that first_name has the values 'Penelope' or 'Nick' or 'Ed'. (Use the IN operator.)
  - Sort the data in all columns in the film table as rental_rate 0.99, 2.99, 4.99 AND replacement_cost 12.99, 15.99, 28.99. (Use the IN operator.)
```

### Project 3
```
Perform the following query scenarios using the dvdrental sample database.

  - List the country names in the country column of the country table, starting with the character 'A' and ending with the character 'a'.
  - List the country names in the country column of the country table that contain at least 6 characters and end with the character 'n'.
  - List the film names in the title column of the film table that contain at least 4 'T' characters, regardless of whether they are uppercase or lowercase.
  - Among the data in all columns in the film table, sort the data whose title starts with the character 'C', whose length is greater than 90 and whose rental_rate is 2.99.
```
